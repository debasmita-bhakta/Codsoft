# 🔐 Password Generator

## Overview
A Python application that generates strong and secure passwords based on a user-defined length.

## Features
- Generate random passwords
- Includes letters, numbers, and special characters
- User-defined password length
- Simple command-line interface

## Technologies Used
- Python 3
- random module
- string module

## How to Run
```bash
python Password_Generator.py
```

## Learning Outcomes
- Functions
- Loops
- Randomization
- String Manipulation
