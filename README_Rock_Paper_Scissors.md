# 🎮 Rock Paper Scissors Game

## Overview
An interactive Rock Paper Scissors game built using Python. Play against the computer and track scores until you exit the game.

## Features
- User vs Computer gameplay
- Random computer choices
- Score tracking
- Multiple rounds
- Tkinter GUI

## Technologies Used
- Python 3
- Tkinter
- random module

## How to Run
```bash
python Rock_Paper_Scissors.py
```

## Learning Outcomes
- GUI Development
- Event Handling
- Conditional Statements
- Python Functions
